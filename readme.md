# Setup Typescript

```sh
# install typescript package
$ npm i -D typescript @types/express @types/node

# setup tsconfig.json
$ npx tsc --init

# install serve for dev
$ npm i -D ts-node-dev

# config script using ts-node-dev
# ex. ts-node-dev server.ts
```
